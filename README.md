# vue-component-usage-loader 

A webpack loader to help you write non-dynamic usage examples for your vue 
components.

## Installation

### Yarn 

```
yarn add vue-component-usage-loader --dev
```

### NPM

```
npm install vue-component-usage-loader --save-dev
```
